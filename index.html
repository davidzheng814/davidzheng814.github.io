wallets.mit.edu

MIT Public Wallet Directory
April 2015 - May 2015
“Wallets” is MIT's official website for students and faculty to securely add and share Bitcoin addresses. In preservering Bitcoin's spirit of privacy, I used the SAML protocol to implement single sign-on authentication for users. The website currently holds over 750 MIT Bitcoin addresses and is an integral part of the campus's growing Bitcoin economy.

RoomSort

Next House Real-time Room Selection Tool
May 2015
With 266 dorm rooms at Next House, many are at a loss when it comes time to choose a room for the coming year. RoomSort provides a sorting algorithm to help residents find rooms tailored to their individual preferences. The app updates in real-time, so that users are notified of room availability changes down to the moment they have to decide. I coded this project using Google's cloud scripting platform, so that Nexties have an easy time using and sharing the app.

Go

The Classic Strategy Game Over TCP/IP
May 2015
Steven Hao and I decided to see what we could create through git without exceeding a combined worktime of 20 hours. The result? A fully functional Go game, complete with scoring system, legal moves indicator, computer AI (basic), and online mode to play across TCP/IP connection. We wrote the program using Python's GUI and sockets library, packages neither of us had used before.

Sudoku Solver

Experimenting with Computer Vision
Fall 2014
Sudoku Solver has two main modules, a game solver and a scanner that reads in images of Sudoku puzzles. The solver attempts increasingly advanced strategies and can assess the difficulty of the puzzle. The scanner crops and filters an image before using the k-Nearest Neighbors algorithm to guess the digits in the image.
